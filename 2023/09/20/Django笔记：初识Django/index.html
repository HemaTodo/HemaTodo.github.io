Django安装、创建项目及App

安装Django

# 安装django
pip install django

# 初始化项目
django-admin startproject xxxx

# 初始化应用
python3 manage.py startapp xxxx

# 生成ddl语句
python3 manage.py makemigrations

# 执行ddl语句
python3 manage.py migrate
